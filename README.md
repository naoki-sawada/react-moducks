# React-Moducks

## About
This is the sample project of [React](https://facebook.github.io/react/), [Moducks](https://github.com/moducks/moducks).

## Requirements

* Node.js v8.6.0 or higher

## Usage

After clone this project, on the project root directory,

```
$ npm install 
(or yarn)
$ npm start
```
